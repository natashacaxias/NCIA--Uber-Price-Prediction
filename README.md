#  NCIA – Uber Price Prediction

**Previsão de Preços de Corridas Uber**  
Aplicação interativa desenvolvida pela **Equipe A (Vespertino)** da **FPF Tech / NCIA**, em parceria com a **Foxconn**, para apresentação na **1ª Amostra Tech AI (2025)**.

O projeto utiliza **Machine Learning** para prever o preço de corridas Uber a partir de variáveis como distância, duração, demanda (surge multiplier) e categoria do serviço, integrando análise exploratória, modelos comparativos e um simulador de previsão.

---

##  Objetivos

- Aplicar **algoritmos de regressão supervisionada** para prever o preço de corridas Uber.  
- Explorar atributos de **mobilidade urbana** (tempo, clima, distância e demanda).  
- Demonstrar o uso do **Streamlit** como ferramenta de apresentação de projetos de IA.  
- Promover **transparência e previsibilidade** nos modelos de precificação dinâmica.  

---

##  Tecnologias Utilizadas

- **Python 3.10+**
- **Streamlit** – Interface interativa
- **Pandas / NumPy** – Manipulação e limpeza de dados
- **Scikit-learn** – Modelos de Regressão e Avaliação
- **Matplotlib / Seaborn** – Visualizações
- **CSS customizado** – Tema institucional FPF Tech / NCIA

##  Como Executar Localmente

### Clonar o repositório
```
git clone https://github.com/<seu-usuario>/NCIA--Uber-Price-Prediction.git
cd NCIA--Uber-Price-Prediction
```

### Instalar dependências
```
pip install -r requirements.txt
```

### Executar o aplicativo
```
streamlit run app.py
```
